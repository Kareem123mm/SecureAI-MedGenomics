### **K-mer Extraction Algorithm**

```python
def extract_kmers(sequence, k=3):
    """
    Extract all k-mers from a DNA sequence
    
    Example:
    sequence = "ATCGATCG"
    k = 3
    
    Process:
    Position 0-2: ATC
    Position 1-3: TCG
    Position 2-4: CGA
    Position 3-5: GAT
    Position 4-6: ATC (duplicate)
    Position 5-7: TCG (duplicate)
    
    Result: ['ATC', 'TCG', 'CGA', 'GAT', 'ATC', 'TCG']
    Unique k-mers: {'ATC', 'TCG', 'CGA', 'GAT'}
    Count: 6 total, 4 unique
    """
    
    kmers = []
    for i in range(len(sequence) - k + 1):
        kmer = sequence[i:i+k]
        kmers.append(kmer)
    
    return kmers
```

### **GC Content Calculation**

```python
def calculate_gc_content(sequence):
    """
    Calculate GC percentage
    
    Example:
    sequence = "ATCGATCGGGCC"
    
    Count:
    A: 2
    T: 2
    C: 4
    G: 4
    Total: 12
    
    GC = (C + G) / Total * 100
       = (4 + 4) / 12 * 100
       = 8 / 12 * 100
       = 66.67%
    
    Interpretation:
    - Human genome: typically 40-60%
    - Bacterial: typically 30-70%
    - Plant: typically 35-50%
    """
    
    gc_count = sequence.count('G') + sequence.count('C')
    total = len(sequence)
    gc_percentage = (gc_count / total) * 100
    
    return gc_percentage
```

### **Species Prediction**

```python
def predict_species(gc_content, sequence_length):
    """
    Predict organism type from GC content
    
    Rules:
    - High GC (>55%): Likely Human or Mammal
    - Medium GC (40-55%): Likely Bacterial
    - Low GC (<40%): Likely Plant or Fungal
    
    Also considers:
    - Sequence length (human chromosomes are longer)
    - K-mer diversity
    """
    
    if gc_content > 55 and sequence_length > 1000:
        return "Human"
    elif 40 <= gc_content <= 55:
        return "Bacterial"
    elif gc_content < 40:
        return "Plant"
    else:
        return "Unknown"
```
